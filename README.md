# 记忆小屋欢迎你！


## 1. 准备工作
安装gitbook-cli，运行 `npm install -g gitbook-cli`

node版本是10.14.1，使用nvm，mac也可以使用nvm


## 2. gitbook基本操作
如果新增gitbook插件，在book.json里新增，运行 `gitbook install`

如果需要本地运行，运行 `gitbook serve`

如果需要生成_book，然后发到github page，运行 `gitbook build`
